Since the intention-to-treat (ITT) principle requires computation of causal effects estimate based on the treatment assignment instead of treatment received, the denominator of mortality risk for assigned and unassigned group is sum of assigned patients and unassigned patients, respectively.

nb <- 10
nobs <- nrow(data)
mort_risk_diff <- rep(NA, nb)
set.seed(1)
for(i in 1:nb) {
  bootidx <- sample(c(1:nobs), size = nobs, replace = TRUE)
  boot_data <- data[bootidx, ]
  
  death_as1 <- boot_data %>% 
  filter(dead == 1 & assigned == 1)
  death_as0 <- boot_data %>% 
  filter(dead == 1 & assigned == 0)

  mort_as1 <- nrow(death_as1)/nrow(boot_data[boot_data$assigned == 1, ])
  mort_as0 <- nrow(death_as0)/nrow(boot_data[boot_data$assigned == 0, ])
  
  mort_risk_diff[i] <- mort_as1 - mort_as0 
  if (i %% 1 == 0)
    print(i)
}

Therefore, the mortality risk difference between treatment assigned group versus treatment unassigned group is -0.002736425 (95%CI: -0.003924136, -0.001548714). Since this does not cross 0, there is statistically significant evidence that the treatment reduces mortality based on the ITT effect estimate.

The assumption required for identification of the ITT effect include exchangeability where the randomization is independent of any confounders and preserved randomization by estimating the effect by assigned group not by treatment received.

In terms of the validity of these assumptions, the exchangeablity is likely ensured because the treatment (vitamin supplement) or placebo was randomly assigned, blocking the path from unmeasured confounders. As for whether randomization is preserved or not, it is valid since I estimated the effect (mortality risk difference) based on the treatment assignment instead of treatment received. In ITT analysis, if there are many non-adherent participants, the effect estimate would be attracted to null.

Using 2-stage least squares method, the IV estimate is -0.003413226 (95%CI: -0.004892316, -0.001934136). Therefore, using either of these two methods, there is statistically significant evidence that receiving treatment decreases the mortality risk among compliers.

The assumptions required for identification of the IV estimand include 1) exchangeability where the assoiation between IV and outcome is not confounded, 2) relevance where the IV is associated with intermediate variable (treatment received), 3) exclusion criteria where there is no direct causal pathway from IV to outcome, and 4) monotonicity where there is no defiers.

As for the exchangeability, this is valid as the treatment was randomly assigned. Relevance is valid since assignment must be related to the treatment actually received (intermediate variable) due to the nature of RCT. Exclusion criteria is likely valid if the trial was double-blinded since knowing who would benefit from the treatment could violate this assumption. Monotonicity is valid since intervention (intervention) was not available for control group.

The strength of instrument refers to the relevance assumption, which is one of the required assumption for instrumental variable. Therefore, the strength of an instrument refers to how strongly the instrument (tax in the state of residence in 1971 dichotomized at $1.25) is associated with the treatment (smoking cessation). In order to quantify this strength, one way to confirm the association is to fit a regression model with the independent variable of an instrument and the response of the treatment to confirm the association. Also, correlation coefficient is another indicator of association.

ci.estimate <- function(data3a = data3){
  EY1 <- mean(data3a$Pred.Y1.update2, na.rm = TRUE)
  EY0 <- mean(data3a$Pred.Y0.update2, na.rm = TRUE)
  
  D1 <- data3a$exercise/data3a$PS_SL*
    (data3a$wt82_71 - data3a$Pred.Y1.update2) + 
    data3a$Pred.Y1.update2 - EY1
  D0 <- (1 - data3a$exercise)/(1 - data3a$PS_SL)*
    (data3a$wt82_71 - data3a$Pred.Y0.update2) + 
    data3a$Pred.Y0.update2 - EY0
  EIC <- D1 - D0
  # ATE variance
  n <- nrow(data3a)
  varHat.IC <- var(EIC, na.rm = TRUE)/n
  # ATE 95% CI
  ATE.TMLE.CI <- c(ATE.TMLE - 1.96*sqrt(varHat.IC), 
                   ATE.TMLE + 1.96*sqrt(varHat.IC))
  return(ATE.TMLE.CI) 
}

sim.r <-function(samplesize = 500){
 set.seed(123)
 expit <-function(x){exp(x)/(1+exp (x))}
 
 #covariates
 L1 <- runif(n=samplesize, 0, 1)
 L2 <- runif(n=samplesize, 0, 1)
 U <- rbinom(n=samplesize, size = 1, prob = 0.4)
 
 #treatment
 Aprob <- expit(3*L1 - 3*L2 + 3*U)
 A <- rbinom(n=samplesize, size =1, prob=Aprob)
 
 #outcome
 Yprob <- expit(3*A + 3*L1 - 3*L2 + 3*U)
 Y <- rbinom(n = samplesize, size = 1, prob = Yprob)
 dat <- cbind(L1, L2, U, A, Y)
 dat <- data.frame(dat)
 return(dat)
}

E-value can be computed with fairly simple computation as follows:

\[ \text{E-value}= RR +\sqrt{RR (RR-1)}, \]

where \(RR = RR^{obs}_{AY}\), which was computed in the Question 4(a). Therefore, the E-value can be given as follows:

RR_obs <- mean(rr_est)
E_value <- RR_obs + sqrt(RR_obs * (RR_obs -1))
E_value
## [1] 2.992209

The computed E-value is 2.992209. E-value represents the minimum strength of association, on the RR, that unmeasured confounder would have to have with the treatment and outcome to fully explain away a specific association between treatment and outcome, conditional on the measured covariates. This value represents the magnitude of the unmeasured confounder that would be required to produce confounding bias equal to the observed treatment-outcome association.
